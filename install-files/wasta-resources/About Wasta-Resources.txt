Wasta-Resources provides an application menu launcher that will open a file
browser to the /usr/share/wasta-resources folder.  Other packages can
place files, folders, or symlinks in the wasta-resources folder in order
to give users easy access to documentation and reference materials.

See the Wasta-Linux website for more information:
https://www.wastalinux.org/

Ask questions at the Wasta-Linux User Group:
https://groups.google.com/forum/#!forum/wasta-linux-users
